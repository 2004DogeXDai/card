# GitHub 网页上传快速指南

## 当前界面操作步骤（最简单方法）✨

### 方法一：拖拽上传（最简单）⭐⭐⭐

1. **找到右下角的蓝色按钮**
   - 按钮上写着 "拖拽至此上传"（Drag here to upload）

2. **打开你的项目文件夹**
   - 打开文件夹：`d:\剧本杀\反骨-观`

3. **拖拽文件到上传区域**
   - 选中以下文件（可以多选）：
     - `index.html`
     - `README.md`
     - `.gitignore`（如果看不到，需要显示隐藏文件）
   - 将这些文件拖拽到蓝色按钮区域或页面上

4. **填写提交信息**
   - 在页面底部的 "Commit changes" 框中
   - 输入：`Initial commit: 惊喜贺卡`

5. **点击绿色按钮**
   - 点击 "Commit changes" 按钮

### 方法二：点击链接上传

1. **找到 "Quick setup" 区域**
   - 看到 "Get started by creating a new file or uploading an existing file."
   - 点击蓝色链接 **"uploading an existing file"**

2. **上传文件**
   - 点击后会进入上传页面
   - 点击 "choose your files" 或直接拖拽文件
   - 选择项目文件夹中的所有文件：
     - `index.html`
     - `README.md`
     - `.gitignore`

3. **填写提交信息并提交**
   - 在底部输入：`Initial commit: 惊喜贺卡`
   - 点击 "Commit changes"

## 重要提示

### 如果看不到 .gitignore 文件：
1. 在 Windows 文件资源管理器中
2. 点击 "查看" 选项卡
3. 勾选 "隐藏的项目"
4. 现在可以看到 `.gitignore` 文件了

### 需要上传的文件清单：
- ✅ `index.html` - 必需（主页面）
- ✅ `README.md` - 可选（项目说明）
- ✅ `.gitignore` - 可选（但建议上传）
- ⚠️ `部署指南.md` 和 `GitHub上传步骤.md` - 可选（不上传也可以）

### 上传后的下一步：
1. 上传完成后，点击仓库页面顶部的 **"Settings"**（设置）
2. 在左侧菜单找到 **"Pages"**
3. 在 "Source" 部分：
   - Branch 选择：`main`
   - Folder 选择：`/ (root)`
4. 点击 **"Save"**（保存）
5. 等待 1-2 分钟，会显示你的网站地址

## 如果遇到问题

**问题：拖拽没反应？**
- 尝试点击 "uploading an existing file" 链接，然后在打开的页面中上传

**问题：找不到 .gitignore 文件？**
- 这个文件不上传也可以，网站可以正常使用

**问题：上传后看不到文件？**
- 刷新页面（F5），文件应该会显示出来

现在就可以开始上传了！推荐使用方法一（拖拽上传），最简单快捷！🚀
