# 部署指南

## 🏠 本地服务器运行（推荐先测试）

如果你想在本地运行这个项目，让本机作为服务器，有以下几种方法：

### 方法一：使用 Python（最简单，推荐）

1. **确保已安装 Python**
   - 如果没有安装，请访问 [Python官网](https://www.python.org/downloads/) 下载安装
   - 安装时记得勾选 "Add Python to PATH"

2. **启动服务器**
   - 双击运行 `启动服务器.bat` 文件
   - 或者在命令行中执行：
   ```bash
   python -m http.server 8000
   ```

3. **访问网站**
   - 打开浏览器，访问：`http://localhost:8000`
   - 或者访问：`http://127.0.0.1:8000`
   - 在同一局域网的其他设备上，可以使用：`http://你的IP地址:8000`

4. **停止服务器**
   - 在命令行窗口中按 `Ctrl+C`

### 方法二：使用 Node.js

1. **确保已安装 Node.js**
   - 如果没有安装，请访问 [Node.js官网](https://nodejs.org/) 下载安装

2. **启动服务器**
   - 在命令行中执行：
   ```bash
   node 启动服务器.js
   ```

3. **访问网站**
   - 打开浏览器，访问：`http://localhost:8000`

### 方法三：直接打开（仅本地查看）

- 直接双击 `index.html` 文件即可在浏览器中打开
- ⚠️ 注意：这种方式某些功能可能无法正常工作，建议使用服务器方式

### 获取本机IP地址（供其他设备访问）

**Windows 系统：**
```bash
ipconfig
```
查找 "IPv4 地址"，例如：`192.168.1.100`

**在同一局域网的其他设备上访问：**
- 手机/平板：`http://192.168.1.100:8000`
- 其他电脑：`http://192.168.1.100:8000`

⚠️ **防火墙提示：** 如果其他设备无法访问，需要在 Windows 防火墙中允许 Python 或 Node.js 通过防火墙。

## 🌐 GitHub Pages 部署（在线访问）

如果你想将网站部署到互联网上，让任何人都可以访问，可以按照以下步骤操作：

## 第一步：创建 GitHub 仓库

1. 登录你的 GitHub 账号
2. 点击右上角的 "+" 号，选择 "New repository"
3. 填写仓库信息：
   - Repository name: `surprise-card`（或你喜欢的名字）
   - Description: 惊喜贺卡网页
   - 选择 Public（公开仓库才能免费使用 GitHub Pages）
   - **不要**勾选 "Initialize this repository with a README"
4. 点击 "Create repository"

## 第二步：上传代码到 GitHub

### 方法一：使用 GitHub Desktop（推荐新手）

1. 下载并安装 [GitHub Desktop](https://desktop.github.com/)
2. 打开 GitHub Desktop，登录你的 GitHub 账号
3. 点击 "File" -> "Add Local Repository"
4. 选择这个项目的文件夹：`d:\剧本杀\反骨-观`
5. 如果提示需要初始化仓库，点击 "Create a Repository"
6. 在左下角填写提交信息，例如："Initial commit: 惊喜贺卡"
7. 点击 "Commit to main"
8. 点击 "Publish repository" 将代码推送到 GitHub

### 方法二：使用 Git 命令行

打开 PowerShell 或命令提示符，执行以下命令：

```bash
# 进入项目目录
cd "d:\剧本杀\反骨-观"

# 初始化 Git 仓库
git init

# 添加所有文件
git add .

# 提交代码
git commit -m "Initial commit: 惊喜贺卡"

# 添加远程仓库（将 YOUR_USERNAME 替换为你的 GitHub 用户名，REPO_NAME 替换为仓库名）
git remote add origin https://github.com/YOUR_USERNAME/REPO_NAME.git

# 推送到 GitHub
git branch -M main
git push -u origin main
```

### 方法三：直接在 GitHub 网页上传

1. 在 GitHub 仓库页面，点击 "uploading an existing file"
2. 将 `index.html`、`.gitignore`、`README.md` 这三个文件拖拽到页面
3. 在下方填写提交信息，点击 "Commit changes"

## 第三步：启用 GitHub Pages

1. 进入你的 GitHub 仓库页面
2. 点击 "Settings"（设置）标签
3. 在左侧菜单中找到 "Pages"（页面）
4. 在 "Source" 部分：
   - Branch 选择 `main`
   - Folder 选择 `/ (root)`
5. 点击 "Save"（保存）
6. 等待几分钟，GitHub 会生成你的网站链接

## 第四步：访问你的网站

1. 在 Pages 设置页面，你会看到一个绿色的提示框，显示你的网站地址
2. 格式通常是：`https://YOUR_USERNAME.github.io/REPO_NAME/`
3. 点击链接即可访问你的惊喜贺卡！

## 更新网站内容

以后如果你修改了代码：

### 使用 GitHub Desktop：
1. 在 GitHub Desktop 中会显示更改
2. 填写提交信息
3. 点击 "Commit to main"
4. 点击 "Push origin" 推送更改

### 使用命令行：
```bash
git add .
git commit -m "更新描述"
git push
```

推送后，GitHub Pages 会自动更新（可能需要几分钟）。

## 常见问题

**Q: 网站显示 404？**
A: 等待 5-10 分钟，GitHub Pages 需要时间部署。确保仓库是 Public 的。

**Q: 如何修改网站链接？**
A: 网站链接基于仓库名，如果想改变链接，需要重新创建仓库。

**Q: 网站更新后看不到变化？**
A: 清除浏览器缓存或使用无痕模式访问。
