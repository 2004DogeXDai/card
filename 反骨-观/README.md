# 惊喜贺卡 💝

一个充满温馨和惊喜的互动贺卡网页，包含摇曳的花灯、飘落的雪花和可爱的花瓣弹窗。

## 功能特点 ✨

- 🏮 左上角摇曳的红灯笼动画
- ❄️ 全屏飘落的雪花效果
- 🌸 点击弹出的粉色花瓣弹窗
- 🎁 温馨的文字和精美图标
- 💫 流畅的动画效果

## 使用方法 🚀

1. 直接在浏览器中打开 `index.html` 文件
2. 点击页面任意位置查看惊喜
3. 点击花瓣可关闭弹窗
