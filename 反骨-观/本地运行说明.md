# 🚀 本地服务器快速启动指南

## ✅ 服务器已启动！

你的本地服务器现在正在运行中。

## 📍 访问地址

### 在本机访问：
- **http://localhost:8000**
- **http://127.0.0.1:8000**

### 在同一局域网的其他设备访问：
1. 打开命令提示符（CMD）或 PowerShell
2. 输入命令：`ipconfig`
3. 查找 "IPv4 地址"，例如：`192.168.1.100`
4. 在其他设备的浏览器中访问：`http://192.168.1.100:8000`

## 🛑 停止服务器

在运行服务器的命令行窗口中按 `Ctrl+C` 即可停止服务器。

## 🔄 重新启动服务器

### 方法一：使用批处理文件（最简单）
- 双击 `启动服务器.bat` 文件

### 方法二：使用命令行
```bash
cd "d:\剧本杀\反骨-观"
python -m http.server 8000
```

### 方法三：使用 Node.js
```bash
cd "d:\剧本杀\反骨-观"
node 启动服务器.js
```

## ⚠️ 注意事项

1. **防火墙设置**：如果其他设备无法访问，需要在 Windows 防火墙中允许 Python 通过防火墙
2. **保持服务器运行**：服务器运行时，不要关闭命令行窗口
3. **端口占用**：如果 8000 端口被占用，可以修改端口号（例如：8001、8080）

## 📱 手机/平板访问

### 快速步骤：
1. **确保服务器正在运行**（运行 `启动服务器.bat`）
2. **获取IP地址**：运行 `获取IP地址.bat` 或查看服务器启动时显示的IP地址
3. **确保手机和电脑在同一WiFi网络**
4. **在手机浏览器输入**：`http://你的IP地址:8000`

例如：`http://192.168.1.100:8000`

### 详细指南：
- 查看 `手机访问指南.md` 获取完整步骤和故障排除方法
- 如果无法访问，查看 `防火墙配置指南.md` 配置防火墙
